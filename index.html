Inspertor de Elementos

(Voltar)

Clique com o botão direito do mouse no elemento que você deseja inspecionar em qualquer página da web. Um menu suspenso será exibido. Selecione Inspecionar. O painel do Inspetor encontrará o elemento selecionado, realçando o código-fonte dele.

Estrutura Basica do HTML

(Voltar)

Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags:

"html, head, body e atributos".

Falando sobre Tags

(Voltar)

Tags básicas do HTML

  • Html - Define um documento HTML.
  • Body - Define o corpo do documento.
  • br - Insere uma única quebra de linha.
  • hr - Define uma linha horizontal.
  • !-- - Define um comentário.

Atributos das Tags

(Voltar)

Atributos são modificadores dos elementos do HTML

  • class=”NomeDaClasse”
  • id=”NomeDoId”
  • src=”Link ou diretório da mídia"
  • alt=”Texto alternativo”
  • href=”Url”
  • lang=”Linguagem”
  • target=”blank” ou "self"

Textos

(Voltar)

A forma como são apresentados os textos na Web Page

  • h1 a h6 - Define título 1 a título 6.
  • p - Define um parágrafo.
  • s - Riscado
  • u - ublinhado
  • i - Italico
  • b - Negrito
  • blockquote - Quote

Listas Ordenadas e Não Ordenadas

(Voltar)

Listas Ordenadas são criadas com ol e li

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas são criadas com ul e li

  • item A
  • Item B
  • Item C

Links

(Voltar)

O Elemento HTML link especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.
